To execute the program you must have the following files in the same directory:

* config.ini		  :Config file
* CadenaConexion.txt  :File with info to connect to bbdd
* instantclient_12_1  :Oracle instantclient. Should be that version
* plantilla-doc/	  :Documents to each environment to promote
* PreparaIOP.exe	  :Executable

For instantclient_12_1 directory you must download from oracle (http://www.oracle.com/technetwork/database/features/instant-client/index-097480.html):
	* instantclient-basic-nt-12.1.0.2.0.zip (67,410,370 bytes) 
	* instantclient-sqlplus-nt-12.1.0.2.0.zip (805,513 bytes)
